# Belfast Rides Mobile (WIP)

This is the mobile version of [Belfast Rides](https://github.com/mwilson35/Belfast-Rides), a taxi booking platform originally developed as a full-stack web app. This project is a work in progress and focuses on bringing the same functionality to iOS and Android using React Native.

At the moment, it’s a basic React Native app with map integration and early layout components for rider/driver functionality. The backend will be shared with the web app, which handles bookings, user roles, and data storage via a Node.js/Express server and MySQL database.

The plan is to replicate features like:
- Booking rides
- Ride history
- Driver document uploads
- Ride assignment and status tracking

Right now, there’s a map. That’s about it. More features coming as sanity and time allow.

Built using:
- React Native (via Expo)
- React Navigation
- MySQL (Railway)
- Node.js/Express (shared backend)
